# springbootswaggersample
Springboot swagger2 sample

Build:
```
$ buildj
```
Get buildj: https://github.com/jht5945/buildj


Run:
```
$ java -jar build/libs/sprintbootswaggersample.jar
```

Access:<br>
http://localhost:8080/swagger-ui.html
